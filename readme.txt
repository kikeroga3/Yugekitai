【タイトル】
遊撃隊 - 最強神復活之巻 -


【ストーリー】
葛飾之高校の裏山で見つかった古代遺跡。それは≪神墓≫と呼ばれる特異な遺跡であった。
同校の人気アイドル・井ノ頭陽子を密かに愛する斎藤浩壱率いる高校生５人組のパンクロックグループ「Ｕ戟隊」は文化祭ライブで大ブーイングを受けた翌日、体育祭で憎き美少年・田中を陥れた後、新聞部・怒屋の話に触発され、謎の転校生・新井がいたという古代遺跡へと足を運ぶが…‥
古代サイキックホラーアドヴェンチャー「遊撃隊」伝説の再幕が今上がる！


【遊び方】
本ソフトはコマンド選択式のアドベンチャーゲームです。

1.ゲーム開始
ゲームを初めて開始するときは、以下の選択肢が表示されます。どちらか一つを選んでタッチ(クリック)してください。

[English_version]
[日本語版]

☆Point!
　英語テキスト版も選択できるので「遊撃隊」で楽しく愉快に英語のお勉強もできるゾ！
　これがホントのイングリッシュアドベンチャーだネ！

2.ゲームの進め方
ゲームが始まると、場面のグラフィックとメッセージが画面に表示され、ストーリーが語られてゆきます。

メッセージ表示エリアの右下に逆三角形の記号「▽」が表示されているときは、メッセージ表示エリアをタッチ(クリック)すれば先へ進みます。

下記のように選択肢(文字列の始めに数字とコロン「：」が付いている)が表示されているときは、どれか一つの行動を選んでタッチ(クリック)して進めてください。

例)
１：考える
２：命ごいする
３：逃げる

3.ゲームの再開について
ゲームを中断して再開したとき、ストーリーがセーブポイントを通過している場合は、ゲーム再開時に下記の選択肢が表示されます。

[Continue]
[New Game]

最初から始める場合は「New Game」を、続きから始めたい場合は「Continue」をタッチ(クリック)し、その後に表示されるセーブポイントの選択肢を選んでタッチ(クリック)してください。
選択肢は「月日_日時 セーブポイント名」で表示されます。

例)12月31日の19時35分に保存された場合

[1231_1935 最後のマサムネ]
